# Commands

`zs` Zoom selected objects.

`sel`*+element* Add to current selection the specified objects (e.g. `seltext`, `selcrv`, `seldim` or `selleader`)

`CTRL + SHIFT + CLICK` Select a part of group or object (member, face, point, edge)
